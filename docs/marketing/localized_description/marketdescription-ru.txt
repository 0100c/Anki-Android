Заучивание наизусть что-либо с AnkiDroid.

AnkiDroid показывает Вам учебные карточки по специальному алгоритму, прежде чем Вы их забудете. AnkiDroid полностью совместим с программой отложенных повторений Anki (включая синхронизацию), которая доступна для Windows, Linux и MacOS.

Программа помогает изучать кусочки Вашей информации, где бы Вы ни были и когда бы Вы ни захотели. Используйте время простоя в автобусе или очереди в супермаркете  или в любой другой ситуации, когда Вам приходится ждать.

Создавайте свою собственную колоду или скачайте бесплатные колоды, созданные для множества языков или тем (в наличии более 6000).

Материал можно добавлять через настольное приложение Anki Desktop или непосредственно через Ankidroid. Программа поддерживает автоматическое добавление материала, переданного из других программ через намерения (share).

★ Основные возможности:
• поддерживаемые виды контента в картах: текст, изображения, звуки, LaTeX
• алгоритм интервальных повторений (supermemo2 алгоритм)
• функция преобразования текста в речь
• более 6000 готовых колод
• progress widget
• детальная статистика
• синхронизация с AnkiWeb
• открытый исходный код

★ Дополнительные возможности:
• возможность писать ответы (опционально)
• доска (чтобы рисовать)
• редактор/добавление карты
• обозреватель карт
• tablet layout
• импорт существующих коллекций карт (через Anki Desktop)
• интеграция словарей (ColorDict, GoldenDict, Leo.org, Aedict, разнообразные онлайн-словари)
• добавление карт через Намерения от других приложений (например, передать из словаря данные в AnkiDroid, автоматически создав карту)
• поддержка пользовательских шрифтов
• система полного резервного копирования
• настраиваемая навигация через росчерк (скольжение пальцем), нажатие, встряхивание
• множество настроек
• dynamic deck handling
• ночной режим
• 27 локализаций (переводов интерфейса)
• All previous AnkiDroid versions can be downloaded from the website
