Zapamiętasz cokolwiek zechcesz z AnkiDroidem!

AnkiDroid pomaga efektywnie zapamiętywać fiszki wyświetlając je tuż przed ich zapomnieniem. W pełni kompatybilny z desktopową aplikacją Anki (włączając synchronizację), która dostępna jest dla Windows, Linuksa oraz MacOS.

Ucz się czegokolwiek zechcesz, gdziekolwiek zechcesz. Wykorzystaj wolny czas w czasie dojazdów autobusem, w kolejce sklepowej lub kiedykolwiek musisz na coś czekać!

Twórz własne talie lub pobierz talie darmowe, tworzone dla wielu języków oraz dziedzin (dostępnych jest ponad 6000 talii).

Dodawanie materiału za pomocą desktopowego programu Anki lub bezpośrednio w AnkiDroidzie Aplikacja wspiera nawet dodawanie materiału automatycznie z poziomu słownika

★ Podstawowe cechy:
• karty mogą zawierać: tekst, obrazki, dźwięki, LaTeX
• planowane powtórki (algorytm SM2)
• wsparcie dla syntezy mowy
• ponad 6000 gotowych talii
• progress widget
• szczegółowe statystyki
• synchronizacja z AnkiWeb
• otwarte oprogramowanie

★ Dodatkowe możliwości:
• wpisywanie odpowiedzi (opcjonalne)
• tablica do rysowania
• edytor kart
• przeglądarka kart
• tablet layout
• importowanie gotowych plików z kolekcjami (poprzez Anki Desktop)
• integracja ze słownikami (ColorDict, GodenDict, Leo.org, Aedict, różne słowniki online)
• dodawanie kart za pomocą intentów z innych aplikacji, np. słowników
• wsparcie dla czcionek użytkownika
• system kopii zapasowych
• nawigacja poprzez gesty, kliknięcia oraz potrząsanie
• w pełni konfigurowalny
• dynamic deck handling
• tryb nocny
• 27 wersji językowych
• All previous AnkiDroid versions can be downloaded from the website
