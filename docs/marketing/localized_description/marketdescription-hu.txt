Memorizálj bármit az AnkiDroiddal!

Az AnkiDroid segítségével hatékonyan tanulhatsz szókártyákat, mert akkor szedi elő újra őket, mielőtt elfelejtenéd. Teljen kompatibilis és szinkronizálható az Anki nevű késleltetett ismétléses programmal, amely elérhető Windowson, Linuxon és MaxOS-en is.

Tanulj meg akármit, bármikor, bárhol, ha rád tör a tanulhatnék! Használd ki jól a buszozás, sorban állás, vagy másféle várakozás miatt kieső időt!

Készíts saját tanulókártya paklit, vagy tölts le ingyeneseket a több, mint 6000, különböző témájú és nyelvű elérhető pakli közül!

Hozz létre új anyagot az asztali Anki-val, vagy rögtön az AnkiDroiddal! Az alkalmazás támogatja az anyagok automatikus hozzáadását a szótárból!

★ Fő funkciók:
• támogatott szókártya tartalmak: szöveg, kép, hang, LaTeX
• késleltetett ismétlés (supermemo 2 algoritmus)
• szöveg-beszéd integráció
• több mint 6000 előre elkészített pakli
• progress widget
• részletes statisztikák
• AnkiWeb szinkronizálás
• nyílt forráskódú

★ További funkciók:
• válaszok beírása (lehetőség)
• tábla
• kártya szerkesztő/hozzáadó
• kártya böngésző
• tablet layout
• importáld a meglévő gyűjtemény fájlokat (Anki Desktop-pal)
• szótár-integráció (ColorDict, GoldenDict, Leo.org, Aedict, különböző webes szótárak)
• kártyák hozzáadása más programokból (pl. szótárak)
• egyéni betűtípus-támogatás
• teljes biztonsági másolat rendszer
• navigáció elhúzással, érintéssel, rázással
• teljesen testreszabható
• dynamic deck handling
• éjszakai mód
• 27 lokalizáció
• All previous AnkiDroid versions can be downloaded from the website
